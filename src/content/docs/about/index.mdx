## 数据来源与版权声明

### TMDB API 使用说明

本网站使用 **The Movie Database (TMDB)** API 获取影视作品信息，包括但不限于：

- 剧集海报图片
- 作品基本信息（标题、上映日期、评分等）
- 作品分类和标签

#### 1. 数据归属

本网站展示的所有影视作品数据（包括海报、简介、评分等信息）均来源于 **The Movie Database (TMDB)**，并遵循其 API 使用条款。

**TMDB 版权所有 © The Movie Database (TMDB)**

#### 2. API 使用合规性

本网站严格遵守 TMDB API 的使用规范：

- ✅ 仅用于非商业目的的信息展示
- ✅ 正确标注数据来源为 TMDB
- ✅ 遵守 API 请求频率限制
- ✅ 不对 TMDB 数据进行转售或商业化利用
- ✅ 保留 TMDB 原始属性和归因信息

> "所有图片均为 TMDB 或其第三方数据提供者的财产，并受版权保护。这些图片可以通过 TMDB API 在特定条件下用于非商业目的。"

```typescript
// API 端点
const TMDB_BASE_URL = 'https://api.themoviedb.org/3';
const IMAGE_BASE_URL = 'https://image.tmdb.org/t/p';

// 使用的 API 方法
// - GET /movie/popular - 获取热门电影列表
// - 图片尺寸: w342, w500, w780（响应式适配）
```

#### 缓存策略

为了优化性能并减少 API 调用频率，我们实现了多级缓存机制：

- **内存缓存**: 5 分钟 TTL
- **浏览器缓存**: 30 分钟 TTL
- **请求去重**: 防止重复请求相同数据

#### 错误处理

系统实现了完善的错误处理机制：

- 自动重试机制（最多 3 次）
- 友好的错误提示界面
- 降级显示方案
