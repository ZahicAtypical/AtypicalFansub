# 教程指南

这里汇集了使用我们字幕资源的各种教程和指南。

## 📖 基础教程

### 字幕格式说明

我们提供的字幕主要包含以下格式：

| 格式     | 扩展名 | 推荐播放器      | 说明                   |
| -------- | ------ | --------------- | ---------------------- |
| ASS/SSA  | `.ass` | MPC-BE, VLC     | 支持特效的高级字幕格式 |
| SRT      | `.srt` | 几乎所有播放器  | 通用字幕格式           |
| MKV 内嵌 | `.mkv` | 任意 MKV 播放器 | 字幕已封装在视频文件中 |

### 常用播放器推荐

#### Windows

- **MPC-BE**（推荐）：轻量、支持 ASS 特效
- **VLC**：跨平台、功能全面
- **PotPlayer**：功能强大、界面友好

#### macOS

- **IINA**（推荐）：原生 Mac 应用、美观易用
- **VLC**：经典选择
- **MPV**：高度可定制

#### 移动端

- **VLC Mobile**（iOS / Android）
- **MX Player**（Android）
- **nPlayer**（iOS）

## ⚡ 快速上手

1. **下载字幕文件** → 从我们的发布渠道获取
2. **确保视频与字幕同名** → 例如 `EP01.mp4` + `EP01.ass`
3. **放入同一文件夹** → 播放器会自动加载字幕
4. **享受观影** → 调整字体大小、延迟等以获得最佳体验

## ❓ 常见问题

### Q: 字幕显示乱码？

A: 请确认播放器编码设置为 UTF-8 或 Unicode。

### Q: 特效字幕无法正常显示？

A: 推荐使用 MPC-BE 或 MPV 播放器以获得完整体验。

### Q: 如何调整字幕延迟？

A: 大多数播放器支持快捷键调整：

- VLC: `G` / `H` 键
- MPC-BE: `Shift` + `←` / `→` 键
