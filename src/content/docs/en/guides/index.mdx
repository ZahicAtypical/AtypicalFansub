# Guides & Tutorials

Here you can find various guides and tutorials for using our subtitle resources.

## 📖 Basic Guides

### Subtitle Format Guide

Our subtitles mainly include the following formats:

| Format       | Extension | Recommended Players | Description                                      |
| ------------ | --------- | ------------------- | ------------------------------------------------ |
| ASS/SSA      | `.ass`    | MPC-BE, VLC         | Advanced subtitle format with effects support    |
| SRT          | `.srt`    | Almost all players  | Universal subtitle format                        |
| MKV Embedded | `.mkv`    | Any MKV player      | Subtitles already encapsulated in the video file |

### Recommended Players

#### Windows

- **MPC-BE** (Recommended): Lightweight, supports ASS effects
- **VLC**: Cross-platform, full-featured
- **PotPlayer**: Powerful, user-friendly interface

#### macOS

- **IINA** (Recommended): Native Mac app, beautiful and easy to use
- **VLC**: Classic choice
- **MPV**: Highly customizable

#### Mobile

- **VLC Mobile** (iOS / Android)
- **MX Player** (Android)
- **nPlayer** (iOS)

## ⚡ Quick Start

1. **Download subtitle file** → Obtain from our release channels
2. **Ensure video and subtitle have the same name** → e.g. `EP01.mp4` + `EP01.ass`
3. **Place them in the same folder** → The player will automatically load the subtitle
4. **Enjoy your viewing** → Adjust font size, delay, etc. for the best experience

## ❓ FAQ

### Q: Subtitles showing garbled text?

A: Please make sure the player encoding is set to UTF-8 or Unicode.

### Q: Effect subtitles not displaying properly?

A: We recommend using MPC-BE or MPV players for the full experience.

### Q: How to adjust subtitle delay?

A: Most players support shortcut key adjustment:

- VLC: `G` / `H` keys
- MPC-BE: `Shift` + `←` / `→` keys
